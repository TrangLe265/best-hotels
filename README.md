# HOTEL FINDER APP
## About
A React Native application for discovering hotels across various cities in Finland.
Users can search, filter by category, and view detailed information about each hotel. It also suggests activities to do while staying at a hotel/ city.
The app is currently in Finnish due to the provided file hoteldata.json being in Finnish

## Features
- Search Hotels: Find hotels by name or city.
- Browse by Category: Filter hotels by type (Business, Unique, Retreat, Smart Value).
- Nearby Hotels: See hotels near your current location or in Helsinki by default.
- Hotel Details: View hotel info, images, and city activities.
- Responsive UI: Optimized for mobile devices with smooth navigation.

## Built: 
- Built with Expo Snack 

## Folder structure: 
- screens: app's screens
- component: reusuable Ui components
- data: static data. 
    - hoteldata.json: data provided
    - citydata.json: AI-generated activities list of cities
    - hotelcategory.json: hotel categories based on available SOKOS brands: Original, Solo, Break, and Heymo
